# CSS PLAY GROUND

Whatever tried in css both in work and home will go here. Each topic will be inside a folder.
e.g. Flexbox, CSS-Grid etc..

### Prerequisites
* Nodejs
* browser-sync

## Getting Started

Follow the below steps to start a dev server.

```
npm install
```

```
npm start
```
